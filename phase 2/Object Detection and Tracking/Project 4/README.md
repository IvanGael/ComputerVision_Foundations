# Project 4

Create a Python tool that uses optical flow to visualize the movement of objects in a video.

The OpticalFlowVisualizer class initializes with parameters for:
- Shi-Tomasi corner detection to find good features to track
- Lucas-Kanade optical flow calculation
- Random colors for visualization

## Install
```sh
pip install -r requirements.txt
```

## Run
```sh
python main.py
```
